    Getting Started

    Installation

    Get started with PHP Framework in a few simple steps.

    Step 1: Download or Clone

    Get the framework files to your project directory.

    Terminal 
    git clone https://github.com/your-repo/php-framework.git
cd php-framework

    Step 2: Configure Environment

    Set up your environment variables and database configuration.

    .env file 
    APP_DEBUG=true
APP_NAME=MyApp

DB_HOST=localhost
DB_USERNAME=root
DB_PASSWORD=
DB_NAME=my_database
    Note: Create a .env file in the root directory with your configuration.

    Step 3: Configure Web Server

    Point your web server to the project root directory.

    Apache

    Create a .htaccess file in the root directory:

    .htaccess 
    RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php?url=$1 [QSA,L]

    Nginx

    Add this to your Nginx configuration:

    nginx.conf 
    location / {
    try_files $uri $uri/ /index.php?url=$uri&$args;
}

    Project Structure

    Understanding the framework's directory structure.

    Directory Structure 
    php-framework/
├── app/
│   ├── controllers/     # Application controllers
│   ├── models/          # Database models
│   └── views/           # View templates
├── core/                # Framework core files
├── public/              # Public assets (CSS, JS, images)
├── config.php           # Configuration file
├── routes.php           # Route definitions
└── index.php            # Application entry point

    Quick Start

    Create your first route and controller.

    1. Define a Route

    routes.php 
    <?php

return [
    'welcome' => [
        'method' => 'GET',
        'path' => '/',
        'controller' => 'html',
        'action' => 'index',
    ],
];

    2. Create a Controller

    app/controllers/html.php 
    <?php

class Html extends Controller
{
    public function index()
    {
        $this->view('home', [
            'title' => 'Welcome',
            'message' => 'Hello, World!'
        ]);
    }
}

    3. Create a View

    app/views/home.php 
    <h1><?= htmlspecialchars($title ?? 'Home', ENT_QUOTES, 'UTF-8') ?></h1>
<p><?= htmlspecialchars($message ?? '', ENT_QUOTES, 'UTF-8') ?></p>
    That's it! Visit your application in the browser to see it in action.
